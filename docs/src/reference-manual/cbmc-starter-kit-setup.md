# Reference manual

## Name

`cbmc-starter-kit-setup` - Set up CBMC proof infrastructure for a repository

## Synopsis

```
cbmc-starter-kit-setup [-h] [--verbose] [--debug] [--version]
```

## Description

Set up CBMC proof infrastructure for a repository.

## Options

`--help, -h`

`--verbose`

* Verbose output.

`--debug`

* Debugging output.

`--version`

* Display version number and exit.
